使用说明

如何获取 API Token?

  1. 登录你的 Memos 服务器
  2. 进入 设置 → 我的账户
  3. 找到 "Access Tokens" 部分
  4. 点击 "创建" 生成新 token
  5. 复制 token 并粘贴到上方输入框

快捷键

  • Ctrl + Enter: 快速创建 Memo
